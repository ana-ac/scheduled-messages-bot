Scheduled Messages Bot

It is a telegram bot that manages the messages that we want to program to be sent in the future on a certain date or periodically through some instant messaging application such as whatsapp or the telegram itself.

name

@scheduled-messages-bot

commands

  • schedule_message - to schedule a message
  • get_scheduled_messages - get scheduled messages that have not yet been sent
  • get_history - obtain the history of sent messages

Installation and local launch

  1. Clone the repo: git clone git@github.com:ana-ac/scheduled-messages-bot.git
  2. Create .env with the environment variables listed in the .env.example
  3. Run npm install in the root folder
